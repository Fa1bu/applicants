# Приложение для учета абитуриентов
В условиях современной системы образования, характеризующейся высокой конкуренцией между учебными заведениями и увеличением потока поступающих, вопрос эффективной организации приемной кампании выходит на первый план. Традиционные методы учета абитуриентов с использованием бумажных журналов и таблиц Excel зачастую приводят, поэтомиу я представляю вам свое приложение для учета абитуриентов.

## Требования к программно-техническому обеспечению
•	операционная система: Microsoft Windows 7 Service Pack 1 и выше (рекомендуется Windows 10 или Windows 11); <br>
•	процессор: тактовая частота не менее 1 ГГц;<br>
•	оперативная память: не менее 512 МБ (рекомендуется 2 ГБ и более);<br>
•	свободное место на диске: не менее 200 МБ для файлов приложения и базы данных;<br>
•	установленная среда выполнения .NET Framework версии 4.7.2 или выше;<br>
•	установленный компонент Microsoft Access Database Engine 2016 (или совместимый) для работы с файлом .accdb через поставщика OLEDB;<br>
•	разрешение экрана: не менее 1024×768 пикселей (рекомендуется 1920×1080).
